## vite-react-ts-npm-scaffold

#### 安装依赖
```bash
yarn
# or
npm install
```

#### 开发组件（example）
```bash
npm start
# or
yarn start
```

#### 组件打包
```bash
npm run build
# or
yarn build
```

#### 组件发布
```bash
npm run publish
# or
yarn publish
```

## 发布组件到npm
```bash
# 编译项目
npm run build

# 对于私有包和无作用域的包，请使用 
npm publish

# 对于作用域公共包，使用 
npm publish --access=public
```
